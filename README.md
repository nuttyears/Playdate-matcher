# Playdate Matcher

An application that helps families coordinate playdates.

## Features

- **Profile Setup**: Create a profile with your child's name, zip code, and parent contact information
- **Playdate Management**: Create, edit, and delete playdates
- **Playdate Types**: Support for both 1:1 and group playdates
- **Status Tracking**: Track playdates through planning, confirmed, and completed statuses
- **Local Storage**: All data is saved locally in your browser

## Getting Started

Simply open `index.html` in a web browser. No build process or installation required!

The application uses:
- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Babel Standalone (for JSX transformation)

## Usage

1. **First Time Setup**: Enter your child's name, zip code, and phone number
2. **Create Playdates**: Click "New Playdate" to schedule a playdate
3. **Manage Playdates**: Edit, delete, or update the status of your playdates
4. **View Profile**: Click on your child's name in the header to view/edit your profile

## Project Structure

- `index.html` - Single-page application containing all React components and logic
